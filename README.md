# java-was-2022
Java Web Application Server 2022


## 1주차 학습 계획 (4.3 ~ 4.7)
## 종합 목표
### HTTP 학습
- HTTP 메소드의 형식과 기능을 공부
- HTTP Request, HTTP Reponse의 차이를 공부
- HTTP GET 메소드 학습
- HTTP Header, HTTP Body에 대해 공부

### 기타 학습
- MIME 타입 학습
- 자바 입출력 학습
- 토비의 스프링 학습
